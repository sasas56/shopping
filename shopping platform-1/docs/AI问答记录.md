# AI问答记录

## 项目概述
本项目是一个基于 Vue3 + Node.js 的电商购物平台，支持商品浏览、搜索、购物车、下单、订单管理等功能。

- Header.vue：将 `<a>` 标签改为 `<button>`，添加 @click 事件

### 问题7：登录验证功能完善

**用户提问**：

**AI解决方案**：
1. 后端：实现 JWT 认证中间件
2. 前端：添加请求拦截器自动携带 Token
3. 前端：添加响应拦截器处理 Token 失效

5. 后端：完善订单查询接口

**技术实现**：
- JWT Token 有效期：24小时
- 自动添加 Authorization 请求头
- 401/403 错误自动跳转登录页
- 路由 meta.requiresAuth 标记受保护页面

## 技术栈总结

### 前端技术栈
- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router
- **HTTP客户端**：Axios
- **样式**：TailwindCSS

### 后端技术栈
- **运行时**：Node.js
- **框架**：Express
- **认证**：JWT (jsonwebtoken)
- **跨域**：CORS

### 项目结构
```
shopping-platform/
├── src/                          # 前端源码
│   ├── api/                      # API接口定义
│   ├── components/               # 公共组件
│   ├── router/                   # 路由配置
│   ├── stores/                   # Pinia状态管理
│   ├── types/                    # TypeScript类型定义
│   ├── utils/                    # 工具函数
│   └── views/                    # 页面组件
├── backend/                      # 后端源码
│   └── server.js                 # 主服务文件
├── public/                       # 静态资源
│   └── images/                   # 商品图片
└── docs/                         # 项目文档
    ├── 登录验证原理说明.md
    └── 查询功能前后端工作过程说明.md
```
